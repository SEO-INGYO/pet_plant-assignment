## 내용
캡스톤디자인 강의를 수강하면서 진행했던 프로젝트입니다. 앱으로 화분의 상태를 실시간으로 확인하고 원격으로 물을 주거나 RGB LED를 제어하면 스피커에서 화분의 상태를 출력해 주는 스마트 화분을 개발했습니다.
